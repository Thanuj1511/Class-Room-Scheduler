# Class-Room-Scheduler

The project's goal is to provide a thorough Class Scheduler system for universities. 
This system will optimize resource allocation and communication by streamlining the scheduling processes for professors, administrators, and students. 


# Use Case Diagram
In our project Administrators and teachers can create, edit, and delete class schedules
Class schedules contain details such as class name, subject, teacher, time, duration, and location
Students can register for classes, and the system confirms their registration status.
